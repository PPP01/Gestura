# Gestura – Store listing copy (all locales)

Ready-to-paste text for the Chrome Web Store, Edge Add-ons (Microsoft Store), and Firefox AMO listings —
in every locale the extension ships (see `_locales/`). English and German are the canonical,
human-reviewed source texts; the other locales are machine-translated from them and should get a
native-speaker pass before relying on them for a real store submission. The short "Chrome Web Store"
descriptions are copied verbatim from the shipped `extDescription` string in `_locales/<lang>/messages.json`.
Keep the FlowMouse attribution — it is both a courtesy and a GPL requirement.

## English

### Name

### Short description (Edge/AMO, ≤ 250 chars)

### Short description Chrome Web Store (≤ 132 chars)

### Detailed description
Gestura makes browsing faster with smooth, customizable mouse gestures — no data
collection, no tracking, fully open source.

Includes everything FlowMouse does
• Custom gestures — 16 built-in plus unlimited ones you define
• Super drag — drag text, links, or images to trigger actions
• Wheel gestures — hold the right button and scroll to switch tabs
• Rocker gestures — hold one button, click the other for back/forward
• Area select — Shift+drag to open or copy many links at once
• Command chains — run several actions from one gesture
• Clean settings UI with an interactive tutorial

Plus what Gestura adds — headlined by Website Menus
• Website menus — ready-made, fully editable popup menus for the sites you use daily: GitHub, YouTube, Amazon (with country selection), Gmail, Google Maps, Microsoft 365, Facebook, Reddit, Wikipedia and more, every entry with a fitting icon
• Your own menus in seconds — creating a menu for any other website is quick: add entries with icons, set URL patterns (or register the current site with a gesture), done
• One contextual gesture, always the right menu — on GitHub you get Issues & Pull Requests, on YouTube your subscriptions, on Amazon your orders; everywhere else a default Search menu (Google, Brave, Perplexity, DuckDuckGo …), plus a Shopping menu that searches your selection on Amazon & eBay
• Menus that stay yours — customize a predefined menu globally, or load it into a single gesture as a customized copy that keeps inheriting future improvements for untouched entries; private per-gesture menus included
• Quick-search bar — optionally append your favorite searches (Brave, Google, Perplexity …) to the bottom of every menu; the current selection is passed along
• Configurable link opening — same tab, or a new tab left/right/first/last, globally and per menu
• Configurable search engines — add your own text AND image engines, reorder and hide them, with per-locale defaults
• Image search — reverse-search images on the engines you choose
• Per-link JavaScript transforms — reshape the selection with a small sandboxed JS snippet before searching (advanced)

Privacy first
Gestura does not collect browsing history, contains no analytics or ads, and uploads
nothing to third-party servers. Settings stay in your browser (optionally synced by
the browser itself). Full policy: see the privacy policy link.

Open source & credits
Gestura is licensed under GPL-3.0. It is a fork of FlowMouse by Hmily[LCG] & Coxxs —
huge thanks to them for the original extension. If you don't need Gestura's extra
features (website menus, configurable search engines, image search), please
check out and support the original FlowMouse.

Source: https://github.com/PPP01/Gestura

### Ausführliche Beschreibung
Gestura macht das Surfen schneller – mit flüssigen, anpassbaren Mausgesten. Keine
Datensammlung, kein Tracking, vollständig Open Source.

Enthält alles, was FlowMouse kann
• Eigene Gesten — 16 vorkonfiguriert, dazu beliebig viele selbst definierte
• Super-Drag — Text, Links oder Bilder ziehen, um Aktionen auszulösen
• Rad-Gesten — rechte Taste halten und scrollen, um Tabs zu wechseln
• Rocker-Gesten — eine Taste halten, die andere klicken für Zurück/Vor
• Bereichsauswahl — Shift+Ziehen, um viele Links auf einmal zu öffnen oder zu kopieren
• Befehlsketten — mehrere Aktionen aus einer Geste
• Aufgeräumte Einstellungen mit interaktivem Tutorial

Dazu kommt bei Gestura — allen voran die Website-Menüs
• Website-Menüs — fertige, voll editierbare Popup-Menüs für deine täglichen Websites: GitHub, YouTube, Amazon (mit Länderauswahl), Gmail, Google Maps, Microsoft 365, Facebook, Reddit, Wikipedia u. v. m., jeder Eintrag mit passendem Icon
• Eigene Menüs in Sekunden — für jede weitere Website ist ein eigenes Menü schnell erstellt: Einträge mit Icons anlegen, URL-Muster setzen (oder die aktuelle Site per Geste registrieren), fertig
• Eine Kontext-Geste, immer das richtige Menü — auf GitHub Issues & Pull Requests, auf YouTube deine Abos, auf Amazon deine Bestellungen; überall sonst ein Standard-Suchmenü (Google, Brave, Perplexity, DuckDuckGo …), dazu ein Shopping-Menü, das die Auswahl direkt bei Amazon & eBay sucht
• Menüs, die deine bleiben — vordefinierte Menüs global anpassen oder als angepasste Kopie in eine einzelne Geste laden, die für unveränderte Einträge weiterhin Verbesserungen erbt; private Gesten-Menüs inklusive
• Mini-Suchleiste — auf Wunsch deine Lieblingssuchen (Brave, Google, Perplexity …) unten an jedes Menü angehängt; die aktuelle Auswahl wird übergeben
• Konfigurierbares Link-Öffnen — im selben Tab oder in neuem Tab links/rechts/am Anfang/am Ende, global und pro Menü
• Konfigurierbare Suchmaschinen — eigene Text- UND Bild-Suchmaschinen hinzufügen, sortieren und ausblenden, mit sinnvollen Voreinstellungen je Sprache
• Bildersuche — Bilder auf den von dir gewählten Suchmaschinen rückwärtssuchen
• JavaScript-Transformationen pro Link — die Auswahl vor der Suche mit einem kleinen, isolierten JS-Snippet umformen (fortgeschritten)

Gestura erfasst keinen Browserverlauf, enthält keine Analyse- oder Werbe-Codes und
lädt nichts auf fremde Server. Einstellungen bleiben im Browser (optional vom Browser
selbst synchronisiert). Vollständige Richtlinie: siehe Datenschutz-Link.

### الوصف التفصيلي
Gestura يجعل التصفح أسرع بفضل إيماءات فأرة سلسة وقابلة للتخصيص بالكامل — بدون جمع بيانات، بدون تتبع، ومفتوح المصدر بالكامل.

يشمل كل ما يقدمه FlowMouse
• إيماءات مخصصة — 16 إيماءة مدمجة بالإضافة إلى عدد غير محدود تحددها بنفسك
• السحب الفائق (Super Drag) — اسحب نصًا أو رابطًا أو صورة لتفعيل إجراء
• إيماءات العجلة — اضغط مع الاستمرار على الزر الأيمن ومرر للتبديل بين علامات التبويب
• إيماءات الأزرار المزدوجة (Rocker) — اضغط مع الاستمرار على زر وانقر على الآخر للرجوع/التقدم
• تحديد المنطقة — Shift + سحب لفتح أو نسخ روابط عديدة دفعة واحدة
• سلاسل الأوامر — نفّذ عدة إجراءات من إيماءة واحدة
• واجهة إعدادات أنيقة مع برنامج تعليمي تفاعلي

بالإضافة إلى ما يضيفه Gestura — وأبرزه قوائم المواقع
• قوائم المواقع — قوائم منبثقة جاهزة وقابلة للتعديل الكامل للمواقع التي تستخدمها يوميًا: GitHub وYouTube وAmazon (مع اختيار الدولة) وGmail وGoogle Maps وMicrosoft 365 وFacebook وReddit وWikipedia والمزيد، وكل عنصر بأيقونة مناسبة
• قوائمك الخاصة خلال ثوانٍ — إنشاء قائمة لأي موقع آخر سريع: أضف عناصر بأيقونات، حدد أنماط الروابط (أو سجّل الموقع الحالي بإيماءة)، وانتهى الأمر
• إيماءة سياقية واحدة، دائمًا القائمة الصحيحة — على GitHub تحصل على Issues وPull Requests، وعلى YouTube اشتراكاتك، وعلى Amazon طلباتك؛ وفي كل مكان آخر قائمة بحث افتراضية (Google وBrave وPerplexity وDuckDuckGo …)، بالإضافة إلى قائمة تسوق تبحث عن اختيارك في Amazon وeBay
• قوائم تبقى ملكك — خصّص قائمة معدة مسبقًا بشكل عام، أو حمّلها في إيماءة واحدة كنسخة مخصصة تستمر في وراثة التحسينات المستقبلية للعناصر غير المعدّلة؛ وتشمل أيضًا قوائم خاصة لكل إيماءة
• شريط بحث سريع — أضف اختياريًا عمليات البحث المفضلة لديك (Brave وGoogle وPerplexity …) أسفل كل قائمة، مع تمرير التحديد الحالي تلقائيًا
• فتح الروابط بشكل قابل للتخصيص — في نفس التبويب أو في تبويب جديد يسار/يمين/أول/أخير، عالميًا ولكل قائمة على حدة
• محركات بحث قابلة للتخصيص — أضف محركات بحث نصية وصور خاصة بك، وأعد ترتيبها وأخفها، مع إعدادات افتراضية لكل لغة
• بحث الصور — ابحث عكسيًا عن الصور باستخدام محركات البحث التي تختارها
• تحويلات جافاسكريبت لكل رابط — أعد تشكيل التحديد قبل البحث بواسطة مقتطف JS صغير معزول (متقدم)

### Име
Gestura – Жестове с мишката

### Кратко описание (Edge/AMO, до 250 знака)
Жестовете с мишката улесняват живота ти — включително умни менюта, създадени за GitHub, YouTube, Amazon и др. Бързо и 100% поверително, без проследяване, изцяло с отворен код!

### Кратко описание за Chrome Web Store (до 132 знака)
Жестовете с мишката улесняват живота ти — с умни менюта за GitHub, YouTube, Amazon и др. Бързо и 100% поверително!

### Подробно описание
Gestura прави сърфирането по-бързо благодарение на плавни, напълно персонализируеми жестове с мишката — без събиране на данни, без проследяване, изцяло с отворен код.

Включва всичко, което прави FlowMouse
• Персонализирани жестове — 16 вградени плюс неограничен брой, които сами определяте
• Супер плъзгане — плъзнете текст, връзка или изображение, за да задействате действие
• Жестове с колелцето — задръжте десния бутон и превъртете, за да превключвате раздели
• Rocker жестове — задръжте единия бутон, кликнете другия за назад/напред
• Избор на област — Shift + плъзгане, за да отворите или копирате много връзки наведнъж
• Вериги от команди — изпълнете няколко действия с един жест
• Прегледен интерфейс за настройки с интерактивен урок

Плюс това, което добавя Gestura — водено от менютата за сайтове
• Менюта за сайтове — готови, напълно редактируеми изскачащи менюта за сайтовете, които използвате всеки ден: GitHub, YouTube, Amazon (с избор на държава), Gmail, Google Maps, Microsoft 365, Facebook, Reddit, Wikipedia и още, всеки елемент със съответна икона
• Собствени менюта за секунди — създаването на меню за друг сайт е бързо: добавете елементи с икони, задайте URL шаблони (или регистрирайте текущия сайт с жест), готово
• Един контекстуален жест, винаги правилното меню — в GitHub получавате Issues и Pull Requests, в YouTube — вашите абонаменти, в Amazon — вашите поръчки; навсякъде другаде — стандартно меню за търсене (Google, Brave, Perplexity, DuckDuckGo …), плюс меню за пазаруване, което търси избраното в Amazon и eBay
• Менюта, които остават ваши — персонализирайте предварително зададено меню глобално или го заредете в отделен жест като персонализирано копие, което продължава да наследява бъдещи подобрения за непроменените елементи; включени са и лични менюта за отделни жестове
• Лента за бързо търсене — по желание добавете любимите си търсачки (Brave, Google, Perplexity …) в долната част на всяко меню; текущият избор се предава автоматично
• Настройваемо отваряне на връзки — в същия раздел или в нов раздел вляво/вдясно/първи/последен, глобално и за отделно меню
• Настройваеми търсачки — добавяйте собствени текстови и графични търсачки, подреждайте и скривайте ги, с подходящи настройки по подразбиране за всеки език
• Търсене по изображение — обратно търсене на изображения чрез избраните от вас търсачки
• JavaScript трансформации за отделна връзка — преобразувайте избора преди търсене с малък изолиран JS код (за напреднали)

Поверителността на първо място
Gestura не събира история на сърфирането, не съдържа анализи или реклами и не качва нищо на сървъри на трети страни. Настройките остават във вашия браузър (по избор синхронизирани от самия браузър). Пълна политика: вижте връзката към политиката за поверителност.

Отворен код и благодарности
Gestura е лицензиран под GPL-3.0. Той е fork на FlowMouse от Hmily[LCG] и Coxxs — огромни благодарности за оригиналното разширение. Ако нямате нужда от допълнителните функции на Gestura (менюта за сайтове, настройваеми търсачки, търсене по изображение), разгледайте и подкрепете оригиналния FlowMouse.

• Områdevalg — Shift+træk for at åbne eller kopiere mange links på én gang
• Kommandokæder — kør flere handlinger fra én bevægelse
• Overskuelig indstillings-UI med interaktiv vejledning

Plus det, Gestura tilføjer — med webstedsmenuer i front
• Webstedsmenuer — færdige, fuldt redigerbare popup-menuer til de sider, du bruger dagligt: GitHub, YouTube, Amazon (med landevalg), Gmail, Google Maps, Microsoft 365, Facebook, Reddit, Wikipedia med mere, hver post med et passende ikon

• Menuer, der forbliver dine — tilpas en foruddefineret menu globalt, eller indlæs den i én enkelt bevægelse som en tilpasset kopi, der fortsat arver fremtidige forbedringer for uændrede poster; private per-bevægelse-menuer er også inkluderet
• Hurtigsøgningslinje — tilføj eventuelt dine foretrukne søgninger (Brave, Google, Perplexity …) nederst i hver menu; det aktuelle udvalg sendes automatisk med
• Konfigurerbar linkåbning — samme fane eller ny fane til venstre/højre/først/sidst, globalt og pr. menu
• Konfigurerbare søgemaskiner — tilføj dine egne tekst- OG billedsøgemaskiner, omorganiser og skjul dem, med standardindstillinger pr. sprog
• Billedsøgning — omvendt billedsøgning på de søgemaskiner, du vælger
• JavaScript-transformationer pr. link — omform udvalget med et lille sandboxet JS-snippet før søgning (avanceret)

Gestura indsamler ikke browserhistorik, indeholder ingen analyse eller reklamer og uploader intet til tredjepartsservere. Indstillinger forbliver i din browser (valgfrit synkroniseret af selve browseren). Fuld politik: se linket til privatlivspolitikken.

• Super Drag — povucite tekst, poveznice ili slike za pokretanje radnji

• Lanci naredbi — pokrenite više radnji jednom gestom
• Pregledno sučelje postavki s interaktivnim vodičem

• Izbornici web-mjesta — gotovi, potpuno uredivi skočni izbornici za web-mjesta koja svakodnevno koristite: GitHub, YouTube, Amazon (s odabirom države), Gmail, Google Maps, Microsoft 365, Facebook, Reddit, Wikipedia i drugi, svaka stavka s prikladnom ikonom
• Vlastiti izbornici za nekoliko sekundi — izradite izbornik za bilo koje drugo web-mjesto brzo: dodajte stavke s ikonama, postavite URL uzorke (ili registrirajte trenutačno web-mjesto gestom), gotovo
• Jedna kontekstualna gesta, uvijek pravi izbornik — na GitHubu dobivate Issues i Pull Requests, na YouTubeu svoje pretplate, na Amazonu svoje narudžbe; svugdje drugdje zadani izbornik za pretraživanje (Google, Brave, Perplexity, DuckDuckGo …), plus izbornik za kupovinu koji pretražuje vaš odabir na Amazonu i eBayu
• Izbornici koji ostaju vaši — prilagodite unaprijed definirani izbornik globalno, ili ga učitajte u pojedinu gestu kao prilagođenu kopiju koja i dalje nasljeđuje buduća poboljšanja za nepromijenjene stavke; uključeni su i privatni izbornici po gesti
• Traka za brzo pretraživanje — po želji dodajte svoje omiljene tražilice (Brave, Google, Perplexity …) na dno svakog izbornika; trenutačni odabir se prosljeđuje uz to
• Prilagodljivo otvaranje poveznica — u istoj kartici ili u novoj kartici lijevo/desno/prvoj/zadnjoj, globalno i po izborniku
• Prilagodljive tražilice — dodajte vlastite tekstualne i slikovne tražilice, promijenite redoslijed i sakrijte ih, sa zadanim postavkama po jeziku
• Pretraživanje slika — obrnuto pretražujte slike na tražilicama koje odaberete
• JavaScript transformacije po poveznici — preoblikujte odabir malim izoliranim JS isječkom prije pretraživanja (napredno)

### Ayrıntılı açıklama
Gestura, akıcı ve özelleştirilebilir fare hareketleriyle gezinmeyi hızlandırır — veri toplama yok, takip yok, tamamen açık kaynak.

FlowMouse'un yaptığı her şeyi içerir
• Özel hareketler — 16 hazır hareketin yanı sıra istediğiniz kadar kendi tanımladığınız hareket
• Süper sürükleme — eylemleri tetiklemek için metin, bağlantı veya görsel sürükleyin
• Tekerlek hareketleri — sağ düğmeyi basılı tutup kaydırarak sekmeler arasında geçiş yapın
• Rocker hareketleri — bir düğmeyi basılı tutup diğerine tıklayarak geri/ileri gidin
• Alan seçimi — birçok bağlantıyı tek seferde açmak veya kopyalamak için Shift+sürükleyin
• Komut zincirleri — tek bir hareketle birden fazla eylem çalıştırın
• Etkileşimli öğreticiye sahip sade bir ayarlar arayüzü

Gestura'nın eklediği ekstralar — başrolde Web Sitesi Menüleri
• Web sitesi menüleri — her gün kullandığınız siteler için hazır, tamamen düzenlenebilir açılır menüler: GitHub, YouTube, Amazon (ülke seçimiyle), Gmail, Google Maps, Microsoft 365, Facebook, Reddit, Wikipedia ve daha fazlası; her girdi uygun bir simgeyle birlikte
• Saniyeler içinde kendi menünüz — başka herhangi bir web sitesi için menü oluşturmak çok hızlı: simgeli girdiler ekleyin, URL kalıpları belirleyin (veya geçerli siteyi bir hareketle kaydedin), hepsi bu
• Tek bir bağlamsal hareket, her zaman doğru menü — GitHub'da Issues & Pull Requests, YouTube'da abonelikleriniz, Amazon'da siparişleriniz karşınıza çıkar; diğer her yerde varsayılan bir Arama menüsü (Google, Brave, Perplexity, DuckDuckGo …) ve seçiminizi Amazon & eBay'de arayan bir Alışveriş menüsü
• Sizin kalan menüler — önceden tanımlı bir menüyü genel olarak özelleştirin veya tek bir harekete, değiştirilmemiş girdiler için gelecekteki iyileştirmeleri almaya devam eden özelleştirilmiş bir kopya olarak yükleyin; harekete özel gizli menüler de dahildir
• Hızlı arama çubuğu — isteğe bağlı olarak favori aramalarınızı (Brave, Google, Perplexity …) her menünün altına ekleyin; geçerli seçim birlikte iletilir
• Yapılandırılabilir bağlantı açma — aynı sekmede veya yeni sekmede sol/sağ/ilk/son, genel olarak ve menü bazında
• Yapılandırılabilir arama motorları — kendi metin VE görsel arama motorlarınızı ekleyin, sıralayın ve gizleyin; dile göre varsayılanlarla
• Görsel arama — seçtiğiniz motorlarda görselleri ters arayın
• Bağlantı başına JavaScript dönüşümleri — aramadan önce seçimi küçük, korumalı (sandbox) bir JS kod parçasıyla yeniden şekillendirin (gelişmiş)

Gestura прискорює перегляд сторінок завдяки плавним, налаштовуваним жестам миші — без збору даних, без відстеження, повністю відкритий код.

Містить усе, що вміє FlowMouse
• Власні жести — 16 вбудованих плюс необмежена кількість власних
• Супер-перетягування — перетягуйте текст, посилання чи зображення, щоб запускати дії

• Виділення області — Shift+перетягування, щоб відкрити чи скопіювати багато посилань одразу
• Ланцюжки команд — виконуйте кілька дій одним жестом
• Охайний інтерфейс налаштувань з інтерактивним посібником

А також те, що додає Gestura — головне — меню для сайтів
• Меню для сайтів — готові, повністю редаговані спливаючі меню для сайтів, якими ви користуєтесь щодня: GitHub, YouTube, Amazon (з вибором країни), Gmail, Google Maps, Microsoft 365, Facebook, Reddit, Wikipedia та інші, кожен пункт із відповідною іконкою
• Власні меню за лічені секунди — створити меню для будь-якого іншого сайту дуже швидко: додайте пункти з іконками, задайте шаблони URL (або зареєструйте поточний сайт жестом) — готово
• Один контекстний жест — завжди потрібне меню — на GitHub ви отримаєте Issues & Pull Requests, на YouTube — свої підписки, на Amazon — свої замовлення; всюди інде — стандартне меню пошуку (Google, Brave, Perplexity, DuckDuckGo …), а також меню покупок, яке шукає ваш вибір на Amazon і eBay
• Меню, що залишаються вашими — налаштуйте попередньо визначене меню глобально, або завантажте його в окремий жест як налаштовану копію, яка продовжує успадковувати майбутні покращення для незмінених пунктів; приватні меню для окремих жестів також включені
• Панель швидкого пошуку — за бажанням додайте улюблені пошукові системи (Brave, Google, Perplexity …) внизу кожного меню; поточний вибір передається разом із запитом
• Налаштовуване відкриття посилань — у тій самій вкладці або в новій зліва/справа/першою/останньою, глобально та для окремого меню
• Налаштовувані пошукові системи — додавайте власні текстові ТА графічні пошукові системи, змінюйте порядок і приховуйте їх, з типовими значеннями для кожної мови
• Пошук за зображенням — виконуйте зворотний пошук зображень у обраних вами системах
• JavaScript-перетворення для окремих посилань — змінюйте вибір за допомогою невеликого ізольованого JS-скрипту перед пошуком (для досвідчених користувачів)

### Mô tả chi tiết
Gestura giúp duyệt web nhanh hơn với cử chỉ chuột mượt mà, có thể tùy chỉnh — không thu thập dữ liệu, không theo dõi, mã nguồn mở hoàn toàn.

Bao gồm mọi tính năng của FlowMouse
• Cử chỉ tùy chỉnh — 16 cử chỉ có sẵn cộng với số lượng không giới hạn cử chỉ do bạn tự định nghĩa
• Kéo siêu tốc (Super drag) — kéo văn bản, liên kết hoặc hình ảnh để kích hoạt hành động
• Cử chỉ cuộn — giữ nút phải và cuộn để chuyển tab
• Cử chỉ Rocker — giữ một nút, nhấp nút còn lại để quay lại/tiến tới
• Chọn vùng — giữ Shift và kéo để mở hoặc sao chép nhiều liên kết cùng lúc
• Chuỗi lệnh — chạy nhiều hành động chỉ từ một cử chỉ
• Giao diện cài đặt gọn gàng kèm hướng dẫn tương tác

Cùng những gì Gestura bổ sung — nổi bật nhất là Menu website
• Menu website — menu bật lên dựng sẵn, có thể chỉnh sửa hoàn toàn cho các trang web bạn dùng hằng ngày: GitHub, YouTube, Amazon (có chọn quốc gia), Gmail, Google Maps, Microsoft 365, Facebook, Reddit, Wikipedia và nhiều hơn nữa, mỗi mục đều có biểu tượng phù hợp
• Tạo menu riêng chỉ trong vài giây — tạo menu cho bất kỳ trang web nào khác rất nhanh chóng: thêm các mục kèm biểu tượng, đặt mẫu URL (hoặc đăng ký trang hiện tại bằng một cử chỉ), xong
• Một cử chỉ theo ngữ cảnh, luôn đúng menu — trên GitHub bạn nhận được Issues & Pull Requests, trên YouTube là các kênh đã đăng ký, trên Amazon là đơn hàng của bạn; ở mọi nơi khác là menu Tìm kiếm mặc định (Google, Brave, Perplexity, DuckDuckGo …), cùng menu Mua sắm tìm kiếm lựa chọn của bạn trên Amazon & eBay
• Menu vẫn luôn là của bạn — tùy chỉnh một menu dựng sẵn trên toàn hệ thống, hoặc nạp nó vào một cử chỉ riêng dưới dạng bản sao đã tùy chỉnh, bản sao này vẫn tiếp tục kế thừa các cải tiến trong tương lai cho những mục chưa thay đổi; đã bao gồm cả menu riêng tư cho từng cử chỉ
• Thanh tìm kiếm nhanh — tùy chọn thêm các công cụ tìm kiếm ưa thích của bạn (Brave, Google, Perplexity …) vào cuối mỗi menu; lựa chọn hiện tại sẽ được gửi kèm theo
• Cấu hình cách mở liên kết — cùng tab, hoặc tab mới bên trái/phải/đầu tiên/cuối cùng, áp dụng toàn hệ thống hoặc theo từng menu
• Cấu hình công cụ tìm kiếm — thêm công cụ tìm kiếm văn bản VÀ hình ảnh của riêng bạn, sắp xếp lại thứ tự và ẩn chúng, với thiết lập mặc định theo từng ngôn ngữ
• Tìm kiếm hình ảnh — tìm kiếm ngược hình ảnh trên các công cụ bạn chọn
• Chuyển đổi JavaScript theo từng liên kết — định hình lại lựa chọn bằng một đoạn mã JS nhỏ chạy trong sandbox trước khi tìm kiếm (nâng cao)
